How we might organise things here:

  • Manage Backends: enable/disable and configure connections to different backends, like ollama, exllama, locally or remote
  • Manage Models: pull, delete, manage models available at different backends...
  • Manage Agents: be able to search them, create new ones, delete, snapshot them, ask them things...
